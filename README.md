Cocos2dxMrubyPlayerDemo
=======================
```
1. Link your DropBox Account to Cocos2dxMrubyPlayer App.
2. Checkout this repo into DropBox/.../Cocos2dxMrubyPlayer/
  $ cd path-to/DropBox/.../Cocos2dxMrubyPlayer
  $ git clone git@github.com:takeru/Cocos2dxMrubyPlayerDemo.git --separate-git-dir=/Users/takeru/tmp/Cocos2dxMrubyPlayerDemo.git
  (Prevent to sync .git directory by DropBox, --separate-git-dir option is recommended.)
```
